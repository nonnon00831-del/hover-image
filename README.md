# hover-image

Notionに埋め込んで使うことを想定した、  
CSSのみで実装した「ホバーで拡大表示される画像」のサンプルです。

## 概要
このリポジトリは、GitHub Pagesを使って公開し、  
Notionの `/embed` 機能で利用できるシンプルなHTML/CSSサンプルを提供します。

JavaScriptは使用しておらず、  
個人情報の取得や外部通信も行いません。

## できること
- 画像にマウスを乗せると拡大表示される
- CSSのみで実装（JavaScript不使用）
- GitHub Pagesで無料公開可能
- Notionに安全に埋め込み可能

## 使い方
1. このリポジトリをGitHub Pagesで公開します
2. 表示されたURLをコピーします
3. Notionで `/embed` を選択し、URLを貼り付けます
4. 画像にホバーすると拡大表示されます
